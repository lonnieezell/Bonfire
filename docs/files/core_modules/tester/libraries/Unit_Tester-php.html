Unit_Test

An abstract base class that all unit tests should extend from.  Provides unit-testing specific asserts, and database connection and management methods.

Inspired by Toast (http://jensroland.com/projects/toast/), SimpleTest (http://simpletest.org), and FuelCMS (http://getfuelcms.com)

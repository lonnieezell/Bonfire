forgot_password

public function forgot_password()

Allows a user to start the process of resetting their password.  An email is allowed with a special temporary link that is only valid for 24 hours.  This link takes them to reset_password().

reset_password()

public function reset_password($email = '',
$code = '')

Allows the user to create a new password for their account.  At the moment, the only way to get here is to go through the forgot_password() process, which creates a unique code that is only valid for 24 hours.

Parameters

$emailThe email address to check against.
$codeA randomly generated alphanumeric code.  (Generated by forgot_password() ).
